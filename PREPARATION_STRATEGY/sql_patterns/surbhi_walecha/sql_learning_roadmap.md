# SQL Learning Roadmap

## Beginner SQL – Foundations

➡️ SELECT, FROM, WHERE, ORDER BY

➡️ GROUP BY, HAVING, LIMIT

➡️ Aggregate functions

➡️ NULL handling

➡️ String and date manipulation

➡️ Arithmetic operations

➡️ Basic joins

## Intermediate SQL – Business Problem Solving

➡️ Self joins

➡️ CASE statements

➡️ Common Table Expressions (CTEs)

➡️ Window functions

➡️ Subqueries

➡️ Set operations (UNION, INTERSECT)

➡️ Views and temporary tables

➡️ Pivoting and unpivoting data

➡️ Query optimization

## Advanced SQL – System & Analytics Design

➡️ Recursive CTEs

➡️ Indexing strategies

➡️ Data modeling

➡️ Designing datasets for reporting and analytics

➡️ Working with stakeholders to understand requirements
